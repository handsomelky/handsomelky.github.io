еүҚиЁҖ

жң¬зі»еҲ—ж–Үз« дёәеӯҰж ЎиҜҫзЁӢгҖҠж•°еӯ—еӣҫеғҸеӨ„зҗҶгҖӢеёғзҪ®зҡ„дёҖдәӣе°ҸprojectдҪңдёҡ

йҷӨдәҶз»ҷеҮәжҜҸдёӘй—®йўҳзҡ„и§Јзӯ”е’Ңд»Јз ҒпјҢжҲ‘д№ҹдјҡйҷ„дёҠзӣёе…ізҹҘиҜҶзӮ№пјҢд»Ҙж–№дҫҝеҗҺз»ӯеӨҚд№ 

еүҚзҪ®жҰӮеҝө

дҝЎеҸ·йў‘зҺҮдёҺеҷӘеЈ°

дёҖдёӘдҝЎеҸ·йҖҡеёёжҳҜз”ұдҪҺйў‘е’Ңй«ҳйў‘йғЁеҲҶз»„жҲҗ

дҪҺйў‘з»„жҲҗйғЁеҲҶпјҡе№іж»‘жҲ–йҖҗж®өе№іж»‘зҡ„еҢәеҹҹпјҢйӮ»иҝ‘еғҸзҙ дә®еәҰеҖјзӣёдјјгҖӮ

й«ҳйў‘з»„жҲҗйғЁеҲҶпјҡйңҮиҚЎжҖ§ејәзҡ„еҢәеҹҹпјҢйӮ»иҝ‘еғҸзҙ дә®еәҰеҖјдёҚеҗҢпјҢйҖҡеёёеңЁиҫ№зјҳжҲ–еҷӘеЈ°зӮ№еӨ„гҖӮ

дёӢйқўжҳҜдёҖдёӘжҳҺжҳҫзҡ„дҫӢеӯҗ

йӮЈд№ҲжҲ‘д»¬е·Із»ҸзҹҘйҒ“й«ҳйў‘дҝЎеҸ·з»ҸеёёжҳҜеӣҫзүҮдёӯзҡ„еҷӘеЈ°пјҢд№ҹе°ұйЎәзҗҶжҲҗз« ең°иғҪжғіеҲ°дёҖз§ҚеҺ»еҷӘж–№жі•

йӮЈе°ұжҳҜе°ҶеӣҫеғҸдёӯеҷӘеЈ°йӣҶдёӯзҡ„йў‘зҺҮиҝҮж»ӨжҺү

ж·ұе…Ҙз ”з©¶еҺ»еҷӘж–№жі•еүҚпјҢжҲ‘д»¬е…ҲжқҘдәҶи§Јжңүе“Әдәӣеёёи§Ғзҡ„еҷӘеЈ°

  1. зӣҗе’ҢиғЎжӨ’еҷӘеЈ°пјҲSalt-and-Pepper Noiseпјүпјҡ
    • зү№зӮ№пјҡиҝҷз§ҚеҷӘеЈ°е‘ҲзҺ°дёәеӣҫеғҸдёӯйҡҸжңәеҲҶеёғзҡ„й»‘зҷҪеғҸзҙ зӮ№гҖӮ
    • дә§з”ҹеҺҹеӣ пјҡйҖҡеёёз”ұеӣҫеғҸдј ж„ҹеҷЁгҖҒдј иҫ“й”ҷиҜҜжҲ–з”ұдәҺж•°жҚ®жҚҹеқҸзӯүеҺҹеӣ еј•иө·гҖӮ
    • иЎЁзҺ°пјҡеӣҫеғҸдёӯеҮәзҺ°зҡ„й»‘зӮ№пјҲвҖқиғЎжӨ’вҖқпјүе’ҢзҷҪзӮ№пјҲвҖқзӣҗвҖқпјүгҖӮ
  2. и„үеҶІеҷӘеЈ°пјҲImpulse Noiseпјүпјҡ
    • зү№зӮ№пјҡзұ»дјјдәҺзӣҗе’ҢиғЎжӨ’еҷӘеЈ°пјҢдҪҶйҖҡеёёеҸӘеҢ…жӢ¬еҚ•дёҖиүІеҪ©зҡ„еғҸзҙ зӮ№пјҲйҖҡеёёжҳҜзҷҪиүІпјүгҖӮ
    • дә§з”ҹеҺҹеӣ пјҡеҸҜиғҪз”ұеӣҫеғҸдј ж„ҹеҷЁзҡ„ж•…йҡңгҖҒдј иҫ“иҝҮзЁӢдёӯзҡ„й”ҷиҜҜжҲ–е…¶д»–жҠҖжңҜй—®йўҳеҜјиҮҙгҖӮ
    • иЎЁзҺ°пјҡеӣҫеғҸдёӯйҡҸжңәеҲҶеёғзҡ„жҳҺдә®еғҸзҙ зӮ№гҖӮ
  3. й«ҳж–ҜеҷӘеЈ°пјҲGaussian Noiseпјүпјҡ
    • зү№зӮ№пјҡиҝҷз§ҚеҷӘеЈ°зҡ„е№…еәҰйҒөеҫӘй«ҳж–ҜеҲҶеёғпјҲжӯЈжҖҒеҲҶеёғпјүпјҢеҚіеҷӘеЈ°еҖјзҡ„еҲҶеёғе‘Ҳй’ҹеҪўжӣІзәҝгҖӮ
    • дә§з”ҹеҺҹеӣ пјҡеҸҜд»Ҙз”ұеӨҡз§ҚеҺҹеӣ дә§з”ҹпјҢеҢ…жӢ¬дј ж„ҹеҷЁеҷӘеЈ°гҖҒз”өеӯҗз”өи·ҜеҷӘеЈ°гҖҒзғӯеҷӘеЈ°зӯүгҖӮ
    • иЎЁзҺ°пјҡеңЁж•ҙдёӘеӣҫеғҸдёӯе№іеқҮеҲҶеёғпјҢйҖҡеёёзңӢиө·жқҘеғҸжҳҜеӣҫеғҸзҡ„зІ’зҠ¶жҲ–иҪ»еҫ®жЁЎзіҠгҖӮ

з©әеҹҹж»Өжіў

ж•°еӯ—еӣҫеғҸеӨ„зҗҶдёӯзҡ„ж»ӨжіўжҳҜдёҖз§Қз”ЁдәҺеўһејәгҖҒжҒўеӨҚгҖҒе№іж»‘жҲ–жЈҖжөӢеӣҫеғҸдёӯзҡ„зү№е®ҡзү№еҫҒзҡ„жҠҖжңҜгҖӮ

е®ғзҡ„жҖқжғіжҳҜеҲ©з”Ёеӣӣе‘Ёзҡ„еғҸзҙ зӮ№дҝЎжҒҜпјҢеҚіе°ҶеғҸзҙ зӮ№зҡ„еҖјжӣҝжҚўдёәиҜҘеғҸзҙ еҸҠе…¶йўҶеҹҹзҡ„еҮҪж•°еҖј

дё»иҰҒз”ЁйҖ”жңүд»ҘдёӢеҮ дёӘж–№йқўпјҡ

  1. еҺ»еҷӘеЈ°пјҡеӣҫеғҸйҖҡеёёдјҡеӣ дёәеҗ„з§ҚеҺҹеӣ пјҲеҰӮдј ж„ҹеҷЁзјәйҷ·гҖҒдј иҫ“й”ҷиҜҜзӯүпјүеҗ«жңүеҷӘеЈ°гҖӮж»ӨжіўеҷЁеҸҜд»Ҙеё®еҠ©еҺ»йҷӨжҲ–еҮҸе°‘иҝҷдәӣеҷӘеЈ°гҖӮ
  2. й”җеҢ–е’Ңиҫ№зјҳеўһејәпјҡж»ӨжіўеҸҜд»Ҙз”ЁжқҘеўһејәеӣҫеғҸдёӯзҡ„иҫ№зјҳпјҢд»ҺиҖҢдҪҝеӣҫеғҸзңӢиө·жқҘжӣҙжё…жҷ°гҖӮ
  3. е№іж»‘е’ҢжЁЎзіҠпјҡйҖҡиҝҮж»ӨжіўеҸҜд»ҘеҜ№еӣҫеғҸиҝӣиЎҢе№іж»‘еӨ„зҗҶпјҢиҝҷеңЁеҺ»йҷӨеҷӘеЈ°жҲ–йҷҚдҪҺеӣҫеғҸз»ҶиҠӮж–№йқўйқһеёёжңүз”ЁгҖӮ
  4. зү№еҫҒжҸҗеҸ–пјҡзү№е®ҡзұ»еһӢзҡ„ж»ӨжіўеҷЁиғҪеӨҹеё®еҠ©иҜҶеҲ«еӣҫеғҸдёӯзҡ„жҹҗдәӣзү№еҫҒпјҢеҰӮзӣҙзәҝгҖҒеңҶеҪўзӯүгҖӮ
  5. жЈҖжөӢеӣҫеғҸпјҡдҫӢеҰӮжЁЎжқҝеҢ№й…Қ

ж»ӨжіўеҷЁеҸҜд»ҘжҳҜзәҝжҖ§зҡ„пјҢд№ҹеҸҜд»ҘжҳҜйқһзәҝжҖ§зҡ„гҖӮзәҝжҖ§ж»ӨжіўеҷЁеҢ…жӢ¬й«ҳйҖҡж»ӨжіўеҷЁгҖҒдҪҺйҖҡж»ӨжіўеҷЁгҖҒеёҰйҖҡж»ӨжіўеҷЁзӯүпјҢеёёи§Ғзҡ„еҰӮй«ҳж–Ҝж»ӨжіўеҷЁгҖҒеқҮеҖјж»ӨжіўеҷЁгҖӮйқһзәҝжҖ§ж»ӨжіўеҷЁеҲҷеҢ…жӢ¬дёӯеҖјж»ӨжіўеҷЁгҖҒеҸҢиҫ№ж»ӨжіўеҷЁзӯүгҖӮдёҚеҗҢзҡ„ж»ӨжіўеҷЁжңүзқҖдёҚеҗҢзҡ„з”ЁйҖ”е’Ңж•ҲжһңгҖӮ

зәҝжҖ§ж»Өжіў

ж»ӨжіўеҖјжҳҜзӣёйӮ»еғҸзҙ еҖјзҡ„зәҝжҖ§з»„еҗҲпјҢе…·жңүзәҝжҖ§е’Ңз§»дҪҚдёҚеҸҳжҖ§зҡ„зү№зӮ№

зәҝжҖ§пјҡ$filter(f+g)=filter(f)+filter(g)$

з§»дҪҚдёҚеҸҳжҖ§пјҡж»ӨжіўеҷЁеҜ№иҫ“е…Ҙзҡ„з§»дҪҚдҝқжҢҒдёҚеҸҳпјҢ$filter(shift(f)) = shift(filter(f))$

жҲ‘д»¬еҸҜд»Ҙд»ҺдёҖдёӘжңҖз®ҖеҚ•зҡ„ж»Өжіўе…ҘжүӢпјҢеҚіеқҮеҖјж»Өжіў

еңЁеқҮеҖјж»ӨжіўдёӯпјҢжҜҸдёӘж ·жң¬пјҲеҚіеӣӣе‘Ёзҡ„еғҸзҙ зӮ№пјүйғҪжңүзӣёеҗҢзҡ„жқғйҮҚ

йӮЈд№ҲдәӢе®һдёҠпјҢжҲ‘д»¬еҸҜд»ҘжҠҠеҚ·з§ҜзңӢдҪңдёҖдёӘз§»еҠЁеҠ жқғеқҮеҖјж»Өжіў

е…¶дёӯжқғеҖјеәҸеҲ—$a[j]$д№ҹеҸҜд»ҘеҸ«еҒҡж»ӨжіўеҷЁ

еҪ“з„¶еңЁиҝҗз®—ж—¶пјҢзәҝжҖ§ж»ӨжіўиҝҳжңүдёҖдәӣеұһжҖ§

  • дәӨжҚўеҫӢпјҲCommutativeпјүпјҡ$a b = b a$пјҢиҝҷиЎЁзӨәзәҝжҖ§ж»ӨжіўеҷЁеҸҜд»ҘдёҺдҝЎеҸ·д»Ҙд»»ж„ҸйЎәеәҸз»“еҗҲпјҢиҖҢдёҚдјҡеҪұе“ҚжңҖз»Ҳз»“жһңгҖӮеңЁеӣҫеғҸеӨ„зҗҶдёӯпјҢиҝҷж„Ҹе‘ізқҖдёҚз®ЎжҳҜе…ҲеҜ№еӣҫеғҸиҝӣиЎҢж»ӨжіўиҝҳжҳҜе…ҲиҖғиҷ‘еӣҫеғҸдёӯзҡ„жҹҗдёӘзү№е®ҡеҢәеҹҹпјҢе…¶з»“жһңжҳҜзӣёеҗҢзҡ„гҖӮ
  • з»“еҗҲеҫӢпјҲAssociativeпјүпјҡ$a (b + c) = (a b) + (a * c)$пјҢиҝҷиЎЁжҳҺеӨҡдёӘзәҝжҖ§ж»ӨжіўеҷЁеҸҜд»ҘдёІиҒ”дҪҝз”ЁпјҢе®ғд»¬зҡ„з»„еҗҲж•ҲжһңдёҺеә”з”ЁеҚ•дёӘз»јеҗҲж»ӨжіўеҷЁзҡ„ж•ҲжһңжҳҜзӯүеҗҢзҡ„гҖӮдҫӢеҰӮпјҢеҰӮжһңжңүдёүдёӘдёҚеҗҢзҡ„ж»ӨжіўеҷЁиҰҒеә”з”ЁдәҺеӣҫеғҸпјҢдҪ еҸҜд»Ҙе…Ҳи®Ўз®—иҝҷдёүдёӘж»ӨжіўеҷЁзҡ„з»„еҗҲпјҢз„¶еҗҺе°Ҷе…¶еә”з”ЁдәҺеӣҫеғҸпјҢиҝҷдёҺдҫқж¬Ўеә”з”ЁжҜҸдёӘж»ӨжіўеҷЁзҡ„з»“жһңзӣёеҗҢгҖӮ
  • еҲҶй…ҚеҫӢпјҲDistributive over additionпјүпјҡ$a (b + c) = (a b) + (a * c)$пјҢиҝҷж„Ҹе‘ізқҖеҰӮжһңдҪ жңүдёӨдёӘеӣҫеғҸжҲ–дҝЎеҸ·йңҖиҰҒеҗҲе№¶пјҢдҪ еҸҜд»Ҙе…ҲеҜ№жҜҸдёӘеӣҫеғҸзӢ¬з«ӢиҝӣиЎҢж»ӨжіўпјҢз„¶еҗҺеҶҚе°Ҷе®ғд»¬зӣёеҠ пјҢиҝҷдёҺе°ҶиҝҷдёӨдёӘеӣҫеғҸйҰ–е…ҲзӣёеҠ пјҢз„¶еҗҺеҜ№з»“жһңеӣҫеғҸиҝӣиЎҢж»ӨжіўпјҢеҫ—еҲ°зҡ„з»“жһңжҳҜдёҖж ·зҡ„гҖӮ
  • ж ҮйҮҸеӣ еӯҗпјҲScalars factor outпјүпјҡ$k \cdot (a b) = (k \cdot a) b = a * (k \cdot b)$пјҢеңЁж»Өжіўж“ҚдҪңдёӯпјҢеҸҜд»ҘжҸҗеүҚе°Ҷж ҮйҮҸпјҲеҚіеёёж•°еӣ еӯҗпјүдёҺж»ӨжіўеҷЁжҲ–дҝЎеҸ·зӣёд№ҳпјҢжҲ–иҖ…еңЁж»Өжіўж“ҚдҪңеҗҺеҶҚд№ҳд»Ҙж ҮйҮҸпјҢжңҖз»Ҳз»“жһңдёҚдјҡж”№еҸҳгҖӮ
  • жҒ’зӯүе…ғзҙ пјҲIdentityпјүпјҡ$a * e = a \quad \text{where} \quad e = [\ldots, 0, 0, 1, 0, 0, \ldots]$пјҢеңЁзәҝжҖ§ж»Өжіўзҡ„дёҠдёӢж–ҮдёӯпјҢеҚ•дҪҚи„үеҶІпјҲдёҖдёӘдёӯеҝғдёә1пјҢе…¶д»–ең°ж–№дёә0зҡ„ж»ӨжіўеҷЁпјүдёҚдјҡж”№еҸҳеӣҫеғҸпјҢиҝҷж„Ҹе‘ізқҖеӣҫеғҸз»ҸиҝҮиҝҷж ·зҡ„ж»ӨжіўеҷЁеҗҺдҝқжҢҒдёҚеҸҳ

еёёи§Ғж»ӨжіўеҷЁ

  1. ж–№жЎҶж»ӨжіўеҷЁпјҡз®ҖеҚ•дё”дҫҝе®ңзҡ„дҪҺйҖҡж»ӨжіўеҷЁгҖӮ

  2. еёҗзҜ·ж»ӨжіўеҷЁпјҡзәҝжҖ§жҸ’еҖјгҖӮ

  3. й«ҳж–Ҝж»ӨжіўеҷЁпјҡйқһеёёе№іж»‘зҡ„жҠ—й”ҜйҪҝдҪҺйҖҡж»ӨжіўеҷЁгҖӮ

зӣёе…іж»Өжіў

д»ҘдёҠжҸҸиҝ°йғҪжҳҜеҹәдәҺдёҖз»ҙз©әй—ҙдёҠзҡ„ж»ӨжіўпјҢиҖҢеҜ№дәҺеӣҫеғҸеӨ„зҗҶпјҢжҲ‘д»¬йңҖиҰҒз”ЁеҲ°дәҢз»ҙз©әй—ҙзҡ„ж»Өжіў

еңЁзӣёе…іж»ӨжіўдёӯпјҢж»ӨжіўеҷЁд№ҹиў«з§°дёәжЁЎжқҝ(template)жҲ–ж ё(kernel)

  1. еҰӮжһңж»ӨжіўеҷЁеҜ№з§°пјҢеҚіh(u,v)=h(-u,-v), еҲҷеҚ·з§Ҝж»Өжіўе’Ңзӣёе…іж»Өжіўзӯүд»·
  2. е°Ҷзӣёе…іж»ӨжіўеҷЁh(u,v)жҢүдёӯеҝғйҖҶж—¶й’Ҳж—Ӣ иҪ¬180еәҰеҶҚдҪңеҚ·з§ҜпјҢе’Ңзӣёе…іж»Өжіўе…·жңүзӯүд»·ж•Ҳжһң
